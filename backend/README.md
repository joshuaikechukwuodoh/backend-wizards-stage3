# Insighta Labs+ Backend API

This is the core API for Insighta Labs+, built with Hono and Bun, deployed on Vercel.

## 🚀 Features
- **GitHub OAuth 2.0 with PKCE & State**
- **JWT Authentication**
- **Role-Based Access Control**
- **Drizzle ORM + PostgreSQL**
- **Rate Limiting & Logging**

## 🛠️ Setup & Development
1. `cd backend`
2. `bun install`
3. Configure `.env` (see `.env.example`)
4. `bun run dev`

## 🧪 Deployment
Deployed on Vercel as a Serverless Function.
